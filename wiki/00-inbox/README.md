# Inbox 使用规则

这里接收尚未确定长期归属的想法、摘录和待处理问题。新内容可以直接创建，命名以日期或主题为主。

定期将内容分流到 [[10-domains/java/_index]]、[[20-projects/mall-swarm/_index]]、[[30-sources/_index]] 或 [[40-maps/知识库导航]]；无法分流的内容可保留，但应标注原因。
