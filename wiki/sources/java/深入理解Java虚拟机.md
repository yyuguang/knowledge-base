# 深入理解Java虚拟机（第3版）

- **作者**：[[entities/java/人物_周志明|周志明]] — 本书作者，资深 Java 技术专家
- **出版社**：机械工业出版社（华章）
- **ISBN**：978-7-111-64124-7
- **版本**：第3版（基于 JDK 7–12）

---

## 核心要点

1. **五大部分**：走近 Java → 自动内存管理 → 虚拟机执行子系统 → 程序编译与代码优化 → 高效并发
2. **内存管理**：JVM 运行时数据区 + 分代 GC 体系 + 经典/低延迟收集器
3. **执行引擎**：Class 文件结构 → 类加载机制 → 字节码执行
4. **编译优化**：前端语法糖 + 后端 JIT + AOT
5. **并发模型**：JMM + 线程安全 + 锁优化
