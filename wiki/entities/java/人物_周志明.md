# 人物_周志明

## 基本信息

- **职业**：资深 Java 技术专家、程序员
- **代表作**：《深入理解Java虚拟机》（1–3 版）、《深入理解OSGi》
- **网络 ID**：icyfenix
- **活跃领域**：JVM 技术、Java 企业级开发、技术写作

## 主要贡献

- 以系统化方式讲解 JVM 内部原理，是国内 JVM 领域最广为流传的入门到进阶读物
- 书中大量实战案例、代码示例和调优经验，兼具理论与实操
- 第3版（2019）更新至 JDK 12，新增 ZGC/Shenandoah/Graal/Jigsaw 等新特性

## 作品时间线

- **2011**：第1版（基于 JDK 7 早期）
- **2013**：第2版（基于 JDK 7 正式版）
- **2019**：第3版（基于 JDK 7–12，35 万字）
