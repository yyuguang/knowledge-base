# 《大话数据结构》书内索引

## 书目信息与适用读者

- 作者：程杰；ISBN：978-7-3022-5565-9；PDF 首页标注出版日期：2011-06-01。
- 适合学过一门高级语言、需要自学或复习数据结构与算法的学习者。
- 书中代码按 C90 约束编写。变量声明、注释形式和指针传参反映历史兼容性；应提炼稳定的 ADT、状态不变式和算法思想，不能把语法细节误作概念本身。

## 全书学习路径

基础模型 → 线性结构 → 非线性结构 → 查找 → 排序：先学习 [[30-sources/books/大话数据结构/第01章_数据结构绪论]] 的关系模型和 ADT，以 [[30-sources/books/大话数据结构/第02章_算法]] 建立评价语言；再进入 [[30-sources/books/大话数据结构/第03章_线性表]]、[[30-sources/books/大话数据结构/第04章_栈与队列]]、[[30-sources/books/大话数据结构/第05章_串]]；继而学习 [[30-sources/books/大话数据结构/第06章_树]] 与 [[30-sources/books/大话数据结构/第07章_图]]；最后以 [[30-sources/books/大话数据结构/第08章_查找]] 和 [[30-sources/books/大话数据结构/第09章_排序]] 收束。

## 学习建议与已知边界

- 每章先复述 ADT 和约束，再纸笔追踪小例子，最后才迁移到当前语言；复杂算法配合断点观察关键状态。
- 书中重在经典内存结构与教学实现；现代标准库、并发语义、外存工程实现和特定语言性能需要另有来源核验。
