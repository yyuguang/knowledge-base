# 概念_记忆系统

## 定义

记忆系统赋予智能体"记住"和"回忆"的能力。hello-agents 将记忆分为四层，每一层有不同的生命周期和检索策略：

| 记忆类型 | 存储周期 | 容量 | 检索方式 | 用途 |
|----------|----------|------|----------|------|
| **Working** | 分钟~小时 | 有限（如 10 条） | 近因 + 关键词 | 当前对话连贯性 |
| **Episodic** | 天~永久 | 无限 | 向量检索 + 时间衰减 | 历史事件回忆 |
| **Semantic** | 永久 | 无限 | 向量 + 知识图谱 | 概念/知识存储 |
| **Perceptual** | 按需 | 无限 | 跨模态向量检索 | 图像/音频记忆 |

## RAG 核心流程

```
用户问题 → 向量检索 → 相关文档片段 → 拼接到 prompt → LLM 生成回答
```

### RAGTool 完整实现

```python
from hello_agents.tools import RAGTool

rag_tool = RAGTool(
    knowledge_base_path="./knowledge_base",
    collection_name="my_knowledge",
    rag_namespace="default"
)

# 添加知识（支持 PDF/Word/Excel/图片/音频等格式）
rag_tool.execute("add_document", file_path="paper.pdf",
                  chunk_size=1000, chunk_overlap=200)
rag_tool.execute("add_text", text="Python由Guido van Rossum于1991年发布。",
                  document_id="python_intro")

# 语义检索
result = rag_tool.execute("search", query="Python编程语言的历史",
                           limit=3, min_score=0.1)

# LLM 增强问答（RAG 管道自动拼接检索结果）
answer = rag_tool.execute("ask", question="Python是什么时候发布的？",
                           enable_mqe=True, enable_hyde=True)
```

### 两条高级检索策略

**MQE（多查询扩展）**：用 LLM 从原始问题生成多个语义不同的查询变体，提高召回率。

```python
def _prompt_mqe(query, n):
    """生成多样化查询扩展"""
    prompt = [
        {"role": "system", "content": "你是检索查询扩展助手。生成语义等价或互补的多样化查询。"},
        {"role": "user", "content": f"原始查询：{query}\n请给出{n}个不同表述的查询。"}
    ]
    text = llm.invoke(prompt)
    return [ln.strip() for ln in (text or "").splitlines() if ln]

# "什么是大语言模型？" → ["大语言模型的定义", "请解释LLM", "什么是LLM？"]
```

**HyDE（假设文档嵌入）**：先用 LLM 生成假设答案段落，用该段落（而非原始问题）做向量检索，缩小查询-文档语义鸿沟。

```python
def _prompt_hyde(query):
    prompt = [
        {"role": "system", "content": "根据用户问题，写一段可能的答案性段落，用于向量检索。"},
        {"role": "user", "content": f"问题：{query}\n请直接写一段中等长度、包含关键术语的段落。"}
    ]
    return llm.invoke(prompt)
```

### 记忆评分公式

不同记忆类型使用不同的评分融合策略：

| 记忆类型 | 评分公式 |
|----------|----------|
| **Working** | `(vector×0.7 + keyword×0.3) × time_decay × (0.8 + importance×0.4)` |
| **Episodic** | `(vector×0.8 + recency×0.2) × (0.8 + importance×0.4)` |
| **Semantic** | `(vector×0.7 + graph×0.3) × (0.8 + importance×0.4)` |
| **Perceptual** | `(vector×0.8 + recency×0.2) × (0.8 + importance×0.4)` |

### 时间衰减函数

```python
def _calculate_recency_score(timestamp):
    """模拟人类遗忘曲线：24小时半衰期，最低 0.1"""
    age_hours = (datetime.now() - datetime.fromisoformat(timestamp)).total_seconds() / 3600
    return max(0.1, math.exp(-0.1 * age_hours / 24))
```
