# 知识库系统

- [[90-system/architecture/知识库架构骨架]] — 当前目录、元数据和迁移规则。
- `templates/` — 新页面模板。
- `workflows/` — 可复用的导入、复盘和维护流程。
- `queries/` — Dataview、搜索和质量检查查询。

所有系统能力都应服务于 [[40-maps/知识库导航]] 和领域 / 项目知识，而不创建新的知识类型目录。
