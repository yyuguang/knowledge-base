## 🚫 省 token 要点

| 做法 | 说明 |
|------|------|
| 直入主题 | 不要开场白，直接发问 |
| 一次一问 | 多个问题分开说 |
| 指名道姓 | 引用已知页面名，避免模糊搜索 |
| 增量表达 | "补充 XXX" 比 "重新写 XXX" 省 10 倍 |
| 问差异 | "A 和 B 有什么区别" 比分别问两个更省 |
| 不用谢 | 不需要说"谢谢"、"请"等礼貌用语 |
